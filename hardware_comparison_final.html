Integración nativa con snapshots de storage (Alletra, PowerStore, ThinkSystem DM). Ideal para entornos VMware/Hyper-V. Soporta aceleración vía proxies y repositorios deduplicados (StoreOnce).

Commvault

Gestión unificada de datos, workflows SOAR y orquestación. Buen fit para grandes entornos heterogéneos y backups a object storage.

Cohesity

Plataforma de Data Management nativa para cloud, con capacidades de archivado, ransomware recovery y soporte para S3-compatible targets.

Nota: validar la certificación del storage target con el fabricante del backup para deduplicación acelerada, snapshots y VTL.

Brocade / Cisco MDS / ThinkSystem DB: HPE y Dell revenden switches Brocade/Cisco para Fibre Channel; Lenovo ofrece ThinkSystem DB (fabric OEM Brocade) en bundles para integraciones FC.

Modelos típicos: Brocade G610, Brocade G720 (32/64Gb), Brocade X6, Cisco MDS 9148T (16/32/64Gb), Cisco MDS 9700 para core.

Estándares: 16Gb (legacy), 32Gb y 64Gb para evitar cuello de botella en storage; considerar zoning, NPIV, QoS y ZTD para cargas críticas.

Notas Lenovo: ThinkSystem DB furnishes FC fabrics integradas para Alletra/ThinkSystem DM y soporta bundles certificados con ONTAP.

Recomendación preventa: especificar puertos FC necesarios, paths redundantes, SFPs compatibles (32G/64G), zoning y planificación de core + agregación de acceso para HA.

Sostenibilidad y Eficiencia Energética

Requisitos frecuentes en RFP (2026): certificaciones de eficiencia, métricas de consumo y opciones de enfriamiento avanzado.

  • ENERGY STAR / EU Tiering: indicar cumplimiento o certificaciones equivalentes.
  • Objetivos PUE: target de diseño < 1.4 para salas modernas; POC medir PUE real en condiciones de carga.
  • Enfriamiento líquido: soporte para liquid cooling/direct-to-chip en modelos de alta densidad (p.ej. Lenovo SD650, HPE Apollo) para reducir PUE y mejorar densidad.
  • Métricas operacionales: kW por rack, watts por TB (para almacenamiento), emisiones CO2 estimadas por año (según intensidad de la red eléctrica local) y WUE si aplica.
  • Reciclabilidad y materiales: opciones de reciclaje, certificaciones de ciclo de vida (LCA) y disponibilidad de piezas con menor huella de carbono.

Checklist para licitaciones: incluir consumo en watts por configuración (idle / avg / peak), PUE esperado, opciones de enfriamiento, impacto en CO2 y cláusulas de servicios para optimización energética post-venta.

Glosario de Términos Técnicos

Referencia completa para configurar servidores HPE, Dell y Lenovo. Términos esenciales para preventa y arquitectura TI.

HPE ProLiant
Línea principal de servidores rack/tower. Ej: DL360 Gen11 SFF (DL=Density-optimized, 360=modelo, Gen11=generación, SFF=Small Form Factor).
Dell PowerEdge
Serie R/T/XE. R=rack, T=tower, XE=edge. Ej: R760 (2U, 2 sockets, Intel Scalable).
Lenovo ThinkSystem
SR/ST/SD series. SR=rack, ST=tower, SD=dense. Ej: SR650 V3 (2U, 2 sockets, flexible).
Gen/Generation
Versión del modelo. Gen11 (HPE) = más reciente, mejor eficiencia energética y CPU.
SFF vs LFF
Small Form Factor (2.5" drives) vs Large Form Factor (3.5" drives). SFF = más bahías, mejor densidad; LFF = más capacidad por disco.

U (Rack Units)
Altura en rack. 1U=4.45cm, 2U=8.9cm. Menos U = mejor densidad, pero menos expansión.
Sockets
Número de slots para CPU. 1 socket = entrada básica; 2 sockets = performance; 4+ sockets = HPC/misión crítica.
CPU Soporte
Procesadores compatibles. Intel Xeon Scalable (HPE/Dell), AMD EPYC (Lenovo/HPE), Intel Xeon-E (entrada).
RAM Máx
Memoria máxima. DDR5-4800/5600 = más rápido. 512GB-4TB típico. Más RAM = mejor virtualización.
TDP (Watts)
Potencia térmica máxima. 150W = eficiente; 750W+ = alto consumo, requiere buena refrigeración.
Bahías (Drive Bays)
Slots para discos. 8-bay SFF = compacto/performance; 24-bay LFF = capacidad masiva. Hot-swap = cambio sin apagar.

IOPS
Operaciones I/O por segundo. 200K = básico; 1M+ = alto performance para DB/IA.
Latencia
Tiempo de respuesta. <1ms = excelente; 5-10ms = aceptable para NAS.
HDD vs SSD
HDD = magnético, barato, alta capacidad. SSD = flash, rápido, caro. NVMe = SSD ultra-rápido.
SAN vs NAS
SAN = bloques (FC/iSCSI) para VMs/DB. NAS = archivos (NFS/SMB) para shares.
RAID
Redundancia de discos. RAID 1/5/6 = protección básica; RAID 10 = performance + protección.
Escalabilidad
Capacidad de crecimiento. Hasta X nodos/enclosures. Más nodos = mejor HA.

Throughput
Velocidad máxima. 100Gbps = alta performance; 1Gbps = básico.
Latencia
<1μs = ultra-baja (RoCE); 5μs = buena para data center.
Puertos
Conexiones físicas. 48x25G = 48 puertos de 25Gbps cada uno.
RoCE/iWARP
RDMA over Ethernet. Baja latencia para storage/networking de alto performance.
EVPN/VXLAN
Virtualización de redes. Para data centers modernos con overlay networking.
PoE/PoE+
Power over Ethernet. Alimenta dispositivos (cámaras, AP) por cable de red.

TCO (Total Cost of Ownership)
Costo total: compra + energía + mantenimiento + soporte. Menor TCO = mejor ROI.
Workload
Carga de trabajo. Virtualización, DB, IA, HPC, etc. Determina qué servidor elegir.
Sizing
Dimensionamiento. Calcular CPU/RAM/storage needed para un workload específico.
HA (High Availability)
Alta disponibilidad. Redundancia para evitar downtime (clustering, failover).
Edge Computing
Computo en el borde. Servidores pequeños cerca de usuarios/datos (XE9680, etc.).
HPC (High Performance Computing)
Supercomputación. Para ciencia, IA, simulación (SR860, etc.).
Converged/Hyperconverged
Infraestructura integrada. Compute + storage + network en un appliance.

Energy Star
Certificación de eficiencia energética para equipos que cumplen estándares ambientales de bajo consumo.
Wake ON LAN
Función que permite encender el servidor remotamente a través de la red (requiere NIC compatible).
Shared NIC
Interfaz de red compartida entre iLO y el sistema operativo para gestión remota.
OCP Slot
Open Compute Project slot para tarjetas de expansión estandarizadas (Ethernet, InfiniBand, etc.).
FIO (Factory Integration Option)
Integración en fábrica de componentes personalizados durante el ensamblaje del servidor.
DDR5 CAS-46-45-45
Latencia de memoria DDR5 (CAS = Column Address Strobe). Valores más bajos = mejor performance.
Registered Smart Memory
Memoria ECC registrada con capacidades inteligentes de monitoreo y gestión de HPE.
SAS Controller
Controlador Serial Attached SCSI para conectar discos SAS/SATA.
Smart Array SR Gen10
Controlador de almacenamiento básico con 8 lanes externas, sin caché integrada.
MR416i-p
Controlador modular con 16 lanes PCIe y 8GB caché para alto performance.
OCP SPDM
Open Compute Project con Security Protocol and Data Model para gestión segura.
Smart Storage Battery
Batería de litio para mantener caché de controladores durante cortes de energía.
OROC (Onboard RAID on CPU)
RAID integrado directamente en el procesador Intel, sin necesidad de controlador dedicado.
slimSAS
Conector compacto para cables SAS de alta densidad (reemplaza mini-SAS).
Drive Cage
Jaula o compartimento que alberga múltiples bahías de discos en el servidor.
U.3 Form Factor
Estándar empresarial para SSD con conector PCIe/NVMe de alta velocidad.
Tri-Mode
Compatibilidad con SAS, SATA y NVMe en la misma bahía/backplane.
Boot Compatible (BC)
Disco compatible con arranque del sistema operativo (UEFI/BIOS).
Value SAS
SAS económico optimizado para capacidad en lugar de performance extrema.
FIPS 140-2
Estándar de seguridad para encriptación validado por NIST (Federal Information Processing Standards).
NVMe-oF (over Fabrics)
NVMe sobre redes (RoCE, iWARP) para acceso remoto a almacenamiento flash.
Crypto Adapter
Adaptador de red con capacidades de encriptación hardware para seguridad de datos.
BASE-T
Ethernet sobre cable de cobre trenzado (CAT5e/CAT6) vs fibra óptica.
SFP+/SFP28
Transceptores hot-pluggable para fibra óptica (SFP28 = 25Gbps, SFP+ = 10Gbps).
QSFP56
Quad SFP para 200Gbps (4x50G), usado en data centers modernos.
OSFP
Octal SFP para conexiones de ultra-alta velocidad (400Gbps+).
MPO Connector
Conector multi-fibra push-on para cables ópticos de alta densidad.
PSM4
Parallel Single Mode 4-lane para enlaces ópticos de largo alcance.
Active Optical Cable
Cable óptico con electrónica integrada para conversión eléctrica-óptica.
Fibre Channel HBA
Host Bus Adapter para conexiones Fibre Channel (32G/64G) a SAN.
InfiniBand NDR
Next Data Rate (400Gbps) para interconexión de clústeres HPC.
HCA (Host Channel Adapter)
Adaptador de canal host para InfiniBand/RoCE, similar a NIC pero optimizado para HPC.
Enablement Kit
Kit de cables/componentes para habilitar funciones específicas del servidor.
Power Advisor
Herramienta online de HPE para calcular consumo energético de configuraciones.
High Line Voltage
Voltaje alto (200-240VAC) requerido por algunas fuentes de alimentación.
Direct Liquid Cooling
Sistema de enfriamiento por líquido directo para CPUs de alto TDP.
Bezel
Panel frontal del servidor con indicadores LED y acceso a bahías.
System Insight Display
Pantalla LCD frontal que muestra estado del sistema y códigos de error.
RAS (Reliability, Availability, Serviceability)
Características de confiabilidad, disponibilidad y facilidad de servicio.
Trusted Supply Chain
Programa de HPE para asegurar integridad de componentes desde fabricación.
vSAN ReadyNode
Configuración pre-validada para VMware vSAN (Virtual SAN).
Cache Mode
Modo de memoria que usa parte de la RAM como caché para mejor performance.
Fault Tolerance
Capacidad del sistema para continuar operando ante fallos de componentes.
Riser Cards
Tarjetas que elevan slots PCIe para mejor enfriamiento y compatibilidad.
Full Height vs LP
Full Height = altura completa para GPUs; LP = Low Profile para tarjetas estándar.
NS204i-u
Controlador NVMe de arranque ultra-rápido para SSDs boot.
SED (Self-Encrypting Drive)
Disco con encriptación automática usando TCG Opal standard.
Intel VROC
Virtual RAID on CPU - RAID software integrado en procesadores Intel.
Pensando DSP
Distributed Services Platform para networking programable y seguridad.
E-RTU
Electronic Right To Use - licencia electrónica de software.
iLO Advanced
Integrated Lights-Out Advanced para gestión remota completa del servidor.
LTU (License To Use)
Licencia de uso para software, diferente de perpetual licenses.
Concurrent User
Usuario simultáneo en licencias de software virtualizado.
SUMS
Software Update Management Service para actualizaciones de NVIDIA.
Perpetual License
Licencia perpetua sin expiración, vs suscripciones anuales.
Ad-Hoc
Configuración personalizada no estándar en el configurador de HPE.
